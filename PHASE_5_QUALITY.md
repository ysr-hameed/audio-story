# PHASE 5: QUALITY ASSURANCE & PUBLISHING

**Purpose:** Verify quality, fix common failure patterns, and prepare for publishing (SEO, thumbnail, audio/video).

**Prerequisites:** Phase 1-4 completed with story written.

## THE 24-POINT VERIFICATION CHECKLIST

Run through EVERY point. If any answer is NO → STOP and FIX before proceeding.

```
[ 1] Genre: Cold psychological horror? (NOT emotional family drama?)
[ 2] NO dead loved ones, NO grief, NO family?
[ 3] NO spouse/children/parents mentioned?
[ 4] Emotional backstory (1,000+ words) covering all 6 template points?
[ 5] **HOOK in first 300 words?** (mystery/discovery appears BEFORE deep backstory)
[ 6] Active protagonist (makes choices with real consequences)?
[ 7] Mechanism shown 5-6 times with genuine variation?
[ 8] All 8 addiction phases VISIBLE with explicit Hindi text?
[ 9] Moral dilemma with TANGIBLE costs on BOTH sides?
[10] All 3 supporting character roles present (Witness, Authority, Keeper)?
[11] Sensory atmosphere every 300-500 words, evolving with story?
[12] **Atmosphere has DREAD, not just description?** (something slightly wrong in each)
[13] Real-world consequences after every major event?
[14] 2-3 coincidence exits/denial phases before acceptance?
[15] Twist that reframes everything + properly foreshadowed?
[16] Twist NOT from the banned list?
[17] Conversational Hindi with emotional fillers throughout?
[18] Short sentences, one idea per line, TTS-friendly flow?
[19] Length: 7,000-10,000 words (aim for ~9,000)?
[20] No digits `[0-9]` or English letters `[a-zA-Z]` remain?
[21] SEO.md has all 4 sections: YouTube Title + Description + Tags + ChatGPT Thumbnail Prompt?
[22] NO "वो" or "ये" used incorrectly in narration?
[23] NO word-by-word / pipe-separated / fragmented sentences?
[24] Every paragraph proofread for spelling, grammar, tense, gender, punctuation?
```

## COMMON FAILURE PATTERNS

Check your story against these known failure patterns:

| Failure | Symptom | Fix |
|---------|---------|-----|
| Story too short | <7,000 words for target | Expand each act. Add more mechanism showings. |
| Backstory too brief | <1,000 words before mystery | Force 1,000+ words of pure character. Cover all 6 template points. |
| Hook too late | Mystery appears after 500+ words | Hook must appear in first 300 words. Restructure opening. |
| Atmosphere is flat | Rich description but no dread | Every atmosphere line must create UNEASE. Add something wrong. |
| Atmosphere drops | First scene has details, then none | Add sight+sound+feel every 300 words. Insert between events. |
| Weak supporting chars | 1 character, 1 appearance, no dialogue | Add 3 characters. Each appears 2-3 times with DIALOGUE. |
| Supporting char missing | Character in template but never appears | Verify every named character appears MINIMUM 2 times. |
| Moral dilemma vague | Both options feel same | Make costs TANGIBLE: lose job vs lose life. Be specific. |
| No real consequences | Everything stays in protagonist's head | After every event ask: "What physically changed?" "Who else knows?" |
| Phase 2-3 skipped | Jumps from curiosity to fear | Force "वाह! कमाल है!" and "बस एक बार और" explicitly. |
| Phase 8 weak | No confrontation | Act 3 MUST have physical confrontation scene. |
| Logical gap | Mechanism contradicts itself | Pick ONE explanation (supernatural OR human). Delete contradictions. |
| Twist not foreshadowed | Twist character barely existed before | Plant 2 clues in Act 1. Make them appear 2-3 times. |
| Twist char appears at end | Only shows up to explain everything | Give early scenes. Establish relationship before reveal. |

## SEARCH & REPLACE FIXES

### Fix 1: Find All Digits

Search your story for: `[0-9]`

Replace with Hindi text:
- `1` → `एक`
- `2014` → `दो हज़ार चौदह`
- `3:11` → `तीन बजकर ग्यारह मिनट`

### Fix 2: Find All English Words

Search your story for: `[a-zA-Z]+`

Common culprits:
- `Room 4` → `कमरा चार`
- `DANGER` → `खतरा`
- `SAMPLE` → `नमूना`

### Fix 3: Find Chandrabindu

Search for: `ाँ`

Replace with: `ां` (simple nasal)

### Fix 4: Find Visarga

Search for: `ः`

Rewrite the sentence to avoid it.

## CREATE SEO.md

Every story needs an `SEO.md` file in its directory with 4 sections.

### Template:

```markdown
# SEO Metadata for [STORY_NAME]

## YouTube Title (Hindi only)
[UNIQUE TITLE — HOOK] | Psychological Horror Story in Hindi

## YouTube Description (Hindi only)
क्या होगा अगर [WHAT IF SCENARIO]?

[3-4 sentence no-spoiler summary introducing protagonist + mystery]

[Atmosphere/setting line]

सुनिए यह डरावनी कहानी — और Subscribe कीजिए।

## YouTube Tags (comma-separated)
[story-specific Hindi tag], psychological horror story hindi, hindi horror story, [occupation] horror story, [setting] horror story hindi, डरावनी कहानी हिंदी, hindi horror audiobook, psychological thriller hindi, scary story hindi, horror story for sleep, [object] horror story, hindi kahani horror, real horror story hindi, horror audiobook hindi, [theme] mystery story, dark horror story hindi

## ChatGPT Thumbnail Prompt
[SPECIFIC SCENE DESCRIPTION]. [MAIN OBJECT]. [PROTAGONIST HINT]. Cinematic horror lighting — 90% shadow, 10% [color] light. Deep contrast, film grain. Hindi text overlay at top center only in Thick bold modern font with subtle black shadow and horror like text: "[STORY-SPECIFIC IMPACTFUL TEXT - NOT GENERIC TIME/DATE]" No text at bottom. Text should match the thumbnail vibe. Clean professional typography, minimal, premium look, must pop in dark mode. 16:9 YouTube thumbnail, ultra-photorealistic.
```

### YouTube Title Rules:

- Max 70 characters
- MUST include main hook or object name
- Add power words: रहस्य, राज, डरावनी, सच, कहानी

```
✅ "खामोश किताबें — एक लाइब्रेरियन का डरावना राज | साइकोलॉजिकल हॉरर स्टोरी हिंदी में"
❌ "एक कहानी किताबों के बारे में" (too vague, no hook)
```

### YouTube Description Rules:

- **Line 1:** Hook question — "क्या होगा अगर..." (what if scenario)
- **Lines 2-4:** Brief no-spoiler summary (3-4 sentences)
- **Line 5:** Atmosphere / setting line
- **Line 6:** Call to action — "सुनिए यह डरावनी कहानी — और Subscribe कीजिए।"
- **NO timestamps, NO English sections, NO social snippets**

### YouTube Tags Rules:

- **15-20 tags**
- **First tag** = story name (Hindi)
- **Mix of Hindi and English** for wider reach
- No hashtags (#), just comma-separated words

### ChatGPT Thumbnail Prompt Rules:

Must include:
1. SPECIFIC location (not "dark room")
2. MAIN object (vial, book, phone, stairs — the mechanism)
3. PROTAGONIST hint (silhouette, partial figure, hand)
4. TEXT overlay — **TOP CENTER ONLY**, max 2 lines, 3-4 words each
5. **TEXT CONTENT — IMPACTFUL + STORY-SPECIFIC:**
   - ❌ NEVER use generic time/date — that's TTS context, not thumbnail hook
   - ❌ NEVER use story name as text
   - ❌ NEVER literal/simple description like "object changes" or "vial rolled"
   - ✅ MUST create CURIOSITY + DANGER — listener should FEAR to click but WANT to click
   - ✅ TEXT Line = dark mystery / unsettling question about the mechanism
   - ✅ Use power words: राज़, खौफ, अपराध, सच, गुनाह, मौत, अंधेरा
   - ✅ Each story gets FRESH text — no copy-paste from previous thumbnails
   - ❌ NO Hindi numerals (३,१७,४०) — use regular digits if needed, but prefer no numbers at all
6. **TEXT STYLE RULES:**
   - ❌ NO glowing, dripping, blood-red, neon, or gothic fonts
   - ✅ Thick bold modern font with subtle black shadow and horror like text
   - ✅ Only at top center — NO text at bottom
7. HORROR aesthetic (cinematic lighting, 90% shadow, dark contrast)
8. End with: "16:9 YouTube thumbnail, ultra-photorealistic, must pop in dark mode"

## CREATE THUMBNAIL

### Option 1: Use AI (Recommended)

1. Copy the ChatGPT Thumbnail Prompt from your SEO.md
2. Go to ChatGPT with DALL-E or another AI image generator
3. Paste the prompt
4. Generate image
5. Download as PNG (1280×720 or higher)
6. Save as `thumbnail.png` in your story directory

### Option 2: Use ImageMagick (Simple Text Overlay)

```bash
convert -size 1280x720 xc:black \
  -fill white -pointsize 60 -font Arial-Bold \
  -gravity north -annotate +0+100 "आपका टेक्स्ट यहाँ" \
  thumbnail.png
```

### Thumbnail Checklist:

```
[ ] Resolution: 1280×720 (16:9 ratio)
[ ] File name: thumbnail.png or thumbnail.jpg
[ ] Placed in story directory: STORIES/[STORY_NAME]/thumbnail.png
[ ] Text is readable on mobile (thick, bold, good contrast)
[ ] Text is at TOP CENTER only (not bottom)
[ ] Dark/atmospheric aesthetic
[ ] Main object visible
[ ] No generic time/date in text
```

## GENERATE AUDIO + VIDEO

### Using the generate.sh Script

The project includes `generate.sh` for TTS and video generation.

**Interactive mode:**
```bash
./generate.sh
```
- Script lists all stories
- You pick the story
- Choose: TTS only, Video only, or Both

**Direct mode:**
```bash
./generate.sh STORY_NAME
```
- Replace STORY_NAME with your story directory name
- Generates both TTS and video automatically

### TTS Configuration

The script uses **Piper TTS** with model: `hi_IN-pratham-medium.onnx`

**Horror Preset Settings:**
```
--length-scale 1.35      # Slow, deliberate pace — builds dread
--noise-scale 0.30       # Natural vocal texture, not robotic
--noise-w 0.30           # Varied speaking rhythm, not monotone
--sentence-silence 0.55  # Long pauses between sentences — creates tension
```

### Output Files

After running generate.sh:
- `STORIES/[STORY_NAME]/audio.wav` — TTS audio file
- `STORIES/[STORY_NAME]/video.mp4` — Final video (1280×720, thumbnail + audio)

### Video Generation

The script automatically:
1. Takes your `thumbnail.png` or `thumbnail.jpg`
2. Combines it with `audio.wav`
3. Outputs `video.mp4` at 1280×720 resolution
4. Ready for YouTube upload
